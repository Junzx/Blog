准备工作

  • 安装(conda)

    • 列出已有的虚拟环境: 

      conda env list

    • 创建新的环境：

      conda create -n py27-torch python=2.7

    • 激活/关闭环境

      source activate py27-torch
source deactivate
    • 安装PyTorch

  • 配置jupyter notebook

安装NeuralCoref

git clone https://github.com/huggingface/neuralcoref.git
cd neuralcoref
pip install -e .
(注：上面e后面有个小数点)
